========================================
XAUUSD AI 交易助手 - 使用说明
========================================

【重要】本程序无需安装 Python 环境！

1. 首次运行前配置：
   - 编辑 .streamlit\secrets.toml 文件
   - 填写您的 OPENAI_API_KEY
   - 或者首次运行时在弹窗中输入

2. 确保 MetaTrader5 已安装并登录
   - 下载地址：https://www.metatrader5.com/
   - 登录您的交易账户
   - 确保 XAUUSD 品种可用

3. 运行程序：
   - 双击 XAUUSD_AI.exe
   - 程序会自动打开浏览器访问 http://localhost:8501
   - 关闭控制台窗口可停止程序

4. 如果出现错误：
   - 检查 MT5 是否正常运行
   - 检查 API Key 是否正确配置
   - 确保网络连接正常（需访问 OpenAI API）

5. API Key 获取方式：
   - 访问 https://platform.openai.com/api-keys
   - 创建新的 API Key
   - 确保账户有余额
